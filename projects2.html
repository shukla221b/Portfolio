📝 GhostNote – Whisper-Powered Tray App for Speech-to-Text

Python | Whisper API | Windows Tray App | pynput | pystray | dotenv

A lightweight system-wide speech-to-text tool built with OpenAI Whisper. It lets users hold a hotkey (Alt + \\), speak, and have the text typed instantly into any active app window.

  • Press & hold Alt + \\ to start recording, release to transcribe and type
  • Minimal setup – just add your OpenAI API key and run
  • Tray icon gives real-time feedback (grey = idle, red = recording)
  • No subscription, no bloat – built for developers and writers
